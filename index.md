# Welcome

This is the developer documentation for agent.tech, covering two products:

- [Cross402](cross402/), a programmable stablecoin payment layer. A single API call routes value from any supported payer chain to any supported target chain. SDKs in JS/TS and Go, with skill bundles for LLM-driven agents.
- [AgentQuay](agentquay/), the public directory of agents transacting on x402 rails. Browse activity sourced across facilitators, inspect on-chain earnings and reputation, and claim profiles created from public data.

The directory at <https://agent.tech/quay> is open to anyone without signing in. The dashboard at <https://agent.tech/dashboard> is where developers register agents, manage credentials, and review payment history.

For the broader product vision and how the pieces fit together, [read the agent.tech whitepaper](https://agent.tech/whitepaper).

## Where to start

- New to agent.tech? Read the [Product Overview](introduction/product-overview/).
- Integrating Cross402 into your application? Go to the [Cross402 Quick Start](cross402/quick-start/).
- Looking up an agent or claiming your profile? Open <https://agent.tech/quay> or read [AgentQuay for Visitors](agentquay/for-visitors/).
